Massachusetts Information Security Standard

This website provides general information and is not intended as legal advice


Summary

This regulation requires an organization to develop, implement, and maintain a written comprehensive information security program that contains administrative, technical, and physical safeguards appropriate to various factors

Subject Entities Applies to all persons and entities that own or license personal information about Massachusetts residents. Does not include state or local government entities
Penalty for Non-Compliance
  • The Attorney General may bring an action to remedy violations and other appropriate relief
  • $100 per data subject affected, up to $50,000 for each instance of improper disposal
How to Comply
  • Develop and maintain a Comprehensive Information Security Program
  • Comply with the computer system security requirements
  • Comply with the Massachusetts data destruction Standard
    • Paper documents must be either redacted, burned, pulverized, or shredded so that personal data cannot practicably be read or reconstructed. Electronic and other non-paper media must be destroyed or erased so that personal information cannot practicably be read or reconstructed
More Information 201 Mass. Code Regs. §§ 17.00-17.05
